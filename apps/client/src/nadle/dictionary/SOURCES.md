# Джерела словників Nadle (UK)

## Hunspell `uk_UA.dic`

- **Репозиторій:** [brown-uk/dict_uk](https://github.com/brown-uk/dict_uk) (MPL-1.1)
- **Призначення:** усі леми, з яких **можна вводити здогади** (`WORDS_UK_*_ALLOWED`), і перевірка, що секретне слово **існує в словнику**.

## Частотний список `raw/uk_opensubtitles_50k.txt`

- **Джерело:** [HermitDave/FrequencyWords](https://github.com/hermitdave/FrequencyWords) — корпус **OpenSubtitles 2016**, файл `content/2016/uk/uk_50k.txt`
- **Ліцензія:** зазвичай **CC BY 4.0** (див. репозиторій автора списків)
- **Призначення:** **порядок «звичайніших»** слів; скрипт `gen-nadle-uk-dictionary.mjs` бере перші ~1000 лем довжин 5/6/7, які **є в Hunspell** і проходять легкий фільтр для секретів. Решта секретів (якщо не вистачає) добирається евристикою з того ж Hunspell.

Список з субтитрів може містити суржик і чужомовні вкраплення; у продакшені залишаються лише слова, **присутні в українському Hunspell**, тобто валідні для гри.
